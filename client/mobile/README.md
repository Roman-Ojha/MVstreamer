# MVstreamer

## Streaming App Using:
1. ReactJs
2. NodeJs 
3. MongoDB (To store auth userDetail)
4. Firebase (To store images, files)
5. Flutter

## How to run Locally:
1. `MVstreamer/client/mobile`
2. Create .env file in Root Directory
3. Get all the environment variables which are include in .env.example
4. Run `flutter run`
